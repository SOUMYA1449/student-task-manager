# 🚀 Student Task Manager (Full Stack Project)

## 📌 Description

The **Student Task Manager** is a full stack web application that allows users to manage their daily tasks efficiently. Users can register, log in, add tasks, and track their progress (pending or completed).

This project is built step-by-step from basic frontend (HTML, CSS, JavaScript) to a complete full stack application using Angular, Spring Boot, and SQL database.

---

## 🎯 Features

* 👤 User Registration & Login
* 📝 Add, Update, Delete Tasks
* 📊 Track Task Status (Pending / Completed)
* 📋 Dashboard to view all tasks
* 🔐 Secure authentication (backend)
* 💾 Data stored in SQL database

## ⚙️ How It Works

1. User registers and logs in
2. Backend verifies user data
3. User can add/manage tasks
4. Data is stored in database
5. Tasks are displayed on dashboard
